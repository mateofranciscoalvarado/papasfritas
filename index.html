HTML Básico

VAMOS A APROBAR EL EXAMEN

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language ) es el componente más básico de la Web. Definir el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web ( CSS ) o la funcionalidad/comportamiento ( JavaScript ).
